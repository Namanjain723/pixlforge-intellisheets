# 📸 How to add your screenshots

Put your PNG screenshots in **this folder** (`assets/screenshots/`) using these **exact
file names**. They appear automatically in the "🖥️ Inside the app" section of the README.

| File name (use exactly) | What to capture |
|-------------------------|-----------------|
| `01_upload_profile.png` | **Upload & Profile** tab — sidebar showing rows/quality/industry + the quality score & summary |
| `02_column_details.png` | **Column Details** tab — a couple of expanded columns with their stats |
| `03_visualize.png`      | **Visualize** tab — the auto 2D dashboard (charts) |
| `04_3d_explorer.png`    | **Visualize** tab — the 🧊 3D Explorer chart (rotated a little looks great) |
| `05_compare.png`        | **Compare Files** tab — the month-over-month chart + delta cards |
| `06_ai_analyst.png`     | **AI Insights** tab — a chat answer with a chart/table |
| `07_forecast.png`       | **Forecast** tab — the trend + forecast chart with the horizon dropdown |
| `08_benchmark.png`      | **Benchmark** tab — the gross-margin vs industry bars |
| `09_export_pdf.png`     | **Export Center** — the download buttons / a page of the branded PDF |
| `10_api_key.png`        | The sidebar **🔑 AI Analyst — API Key** panel (with a green "connected" tick) |

## Tips
- **Width ~1400px** looks best on GitHub. PNG preferred.
- Crop out anything sensitive; the **sample data** (`sample_data/`) is perfect for clean shots.
- After adding them here, **also copy this folder to `public_repo/assets/screenshots/`**
  before pushing the public repo (or just copy the whole `assets/` folder over).

That's it — no code changes needed; the README already points at these names.
